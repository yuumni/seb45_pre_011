## 커밋 규칙

### FE 커밋 규칙
작성 예시 : [분류 메시지] "작업내용"  "번호)" "커밋 내용"
ex) [commit] 로그인 기능 3) 비밀번호 글자수 확인

=> 분류 메시지
commit: 새로운 기능 추가
fix: 버그 수정
docs: 문서 변경
style: 코드 스타일 변경
refactor: 코드 리팩토링
test: 테스트 코드 추가나 수정
chore: 빌드 관련 작업 또는 도구 설정 변경
hotfix : 긴급 코드 수정 필요(전원공지 필수)
